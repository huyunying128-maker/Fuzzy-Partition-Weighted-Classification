# Experiment outputs

This directory is the default location for experiment results generated by the scripts in this repository.

Generated files are not tracked by Git because they may contain large arrays, predictions, figures, and model artifacts. The repository keeps this README so that the directory structure is visible.

Typical outputs include:

- `results.json`
- `partition_history.csv`
- `test_predictions.csv`
- `truncation_results.csv`
- `external_classifier_results.csv`
- `ablation_results.csv`
- summary tables in `.csv` and `.md` formats
- generated figures for centroid and classifier visualization
