# Assets

This directory is reserved for figures used in the repository documentation.
Typical examples include the pipeline diagram, digit centroid prototypes,
standardization examples, and classifier comparison plots.

Recommended contents after running the experiments:

```text
assets/
├── pipeline.png
├── centroid_prototypes.png
├── digit_standardization_example.png
└── classifier_comparison.png
```

Generated experiment figures can be copied here when they are suitable for the
repository README. Raw experimental outputs should remain in `outputs/`.
