# Paper

This directory is reserved for paper-related files, such as the accepted PDF,
BibTeX entry, supplementary notes, and camera-ready materials.

Recommended contents after publication:

```text
paper/
├── accepted_paper.pdf
├── bibtex.txt
└── supplementary_notes.md
```

Large intermediate files, editable source archives, and private submission
materials should not be committed unless they are intended for public release.
